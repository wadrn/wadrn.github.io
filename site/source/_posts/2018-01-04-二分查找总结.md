# 二分查找总结

## 二分查找主要解决的问题

一堆数中找出指定的数

## 条件

数组、有序 （数据结构不能用链表）

## 用二分法精确查找

```javascript
var binarySearch = function(arr,low,high,target){
var mid = parseInt((low+high)/2);
while(low  target){
high = mid-1;
}else{
return mid;
}
mid = parseInt((low+high)/2);
}
return -1;
}
```

## 用二分法界限查找

### 严格界限查找

在有序数组中找到

1. 第一个大于target的值
2. 最后一个小于target的值

```javascript
var binarySearchUpperBound = function(arr,low,high,target){
if(low >high || target >=arr[high]){
return -1;
}
var mid = parseInt((low+high)/2);
while(high >low){
if(arr[mid] > target){
high = mid;  //arr[mid]也有可能是结果
}else{
low = mid+1;
}
mid = parseInt((low+high)/2);
}
return mid;
}
var binarySearchLowerBound =function(arr,low,high,target){
if(low >high || target >=arr[high]){
return -1;

var mid = parseInt((low+high+1)/2);
```
