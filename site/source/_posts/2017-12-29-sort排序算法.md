# 选择排序

## 特点：

1. 运行时间和输入无关 与初始状态无关
2. 数据移动是最少的
3. 不稳定排序（举个例子：序列5 8 5 2 9， 我们知道第一趟选择第1个元素5会与2进行交换，那么原序列中两个5的相对先后顺序也就被破坏了。）

```javascript
//选择排序
var selectSort  = function(arr){
for(var i=0;i=0;j--){
if(arr[j]>item){
arr[j+1] = arr[j];
}else{
break;
}
}
arr[j+1] = item;
}
return arr;
}
```
