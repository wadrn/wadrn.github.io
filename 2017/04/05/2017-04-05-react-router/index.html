React-Router

路由本质就是嵌套的视图组件

browserHistory

功能：监听浏览器地址栏的变化，解析URL并与路由组件进行匹配。还提供了一些方法让你在代码中进行导航。
//跳转到某个路径
browserHistory.push('./some.path');
//回退到先前的地址
browserHistory.goBack()
